        📖 How To Use
        

          
_

          
✕

        

      

      

        
    
          
  1. Open 🔑 Token Vault and paste at least one free API key.
    2. 
          
  2. Get free keys: Groq,
            Gemini,
            OpenRouter.
    3. 
          
  3. Keys are saved only in this browser — never uploaded.
    4. 
          
  4. Open 🌸 Blossom, type a prompt, and click Run pipeline.
    5. 
          
  5. If a provider is rate-limited, Blossom auto-fails over to the next.
    6. 
          
  6. Reorder providers in the vault with ▲ / ▼ to set failover priority.
    7. 
        

        
Full guide: see HOW_TO_USE.md in the repo.
